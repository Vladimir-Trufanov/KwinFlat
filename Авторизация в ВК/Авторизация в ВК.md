### [Как узнать дату регистрации vk?](https://ru.stackoverflow.com/questions/315393/Как-узнать-дату-регистрации-vk)

Есть такой зверь - FOAF (Friend of a Friend). Если введёте такой запрос:

```
http://vk.com/foaf.php?id=1
```

```

```
